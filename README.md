# Vibe Coding Platform

Ultra-granular prompts for enterprise-quality code generation.

##  Phase 1 Complete - The Wizard (All 4 Stages)

### Stage 1: Intent Declaration
Project name + description

### Stage 2: Business Analysis
5 questions: users, features, scale, integrations, auth

### Stage 3: Technical Analysis
- **5 Archetypos**: JWT Auth, Session Auth, Notifications, Payments, File Upload
- **Ultra-Granular Plans**: Tasks 10 min, phased, time-estimated

### Stage 4: Execution Preview
- Complete project summary
- Business requirements review
- Archetypos visualization
- Execution plan with estimates
- Next steps explanation

##  Quick Start

```bash
# Frontend
cd frontend
npm install
npm run dev
# Open http://localhost:3000

# Backend (Phase 2)
cd backend
npm install
npm run start:dev
```

##  Architecture

- **Frontend**: Next.js 14, TypeScript, Tailwind, shadcn/ui
- **Backend**: NestJS, MongoDB, Passport, JWT
- **Docker**: Compose (frontend, backend, mongo, redis)
