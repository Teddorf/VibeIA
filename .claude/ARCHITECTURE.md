## Security Layers

```
┌───────────────────────────────────────────────────────┐
│                    Request Flow                        │
└───────────────────────────────────────────────────────┘
                          │
                          ▼
┌───────────────────────────────────────────────────────┐
│  1. Rate Limiting (Throttler)                         │
│     - 100 requests/minute per IP                      │
└───────────────────────────────────────────────────────┘
                          │
                          ▼
┌───────────────────────────────────────────────────────┐
│  2. CORS Validation                                   │
│     - Whitelist: frontend URLs only                   │
└───────────────────────────────────────────────────────┘
                          │
                          ▼
┌───────────────────────────────────────────────────────┐
│  3. JWT Authentication                                │
│     - Access token (15min) + Refresh token (7d)       │
│     - @Public() decorator for opt-out                 │
└───────────────────────────────────────────────────────┘
                          │
                          ▼
┌───────────────────────────────────────────────────────┐
│  4. Role-Based Access Control                         │
│     - @Roles('admin', 'user') decorator               │
└───────────────────────────────────────────────────────┘
                          │
                          ▼
┌───────────────────────────────────────────────────────┐
│  5. Input Validation (class-validator)                │
│     - DTOs with validation decorators                 │
└───────────────────────────────────────────────────────┘
                          │
                          ▼
┌───────────────────────────────────────────────────────┐
│  6. Business Logic + Data Access                      │
└───────────────────────────────────────────────────────┘
```
